# 달콤교육 부서별 업무 현황 대시보드 기획서

## 개요

달콤교육 내부용 업무 현황 대시보드를 단일 HTML 파일로 제작합니다.
서버 없이 브라우저에서 바로 실행되고, localStorage로 데이터를 영속 저장합니다.
맥북·아이패드 반응형, 한글 UI, 비전공자도 직관적으로 쓸 수 있는 깔끔한 디자인이 핵심입니다.

---

## 파일 구성

| 파일 | 설명 |
|------|------|
| `dashboard.html` | 대시보드 본체 (더블클릭으로 실행) |
| `대시보드_기획.md` | 이 기획서 |

- 외부 서버·설치 불필요 (순수 HTML + CSS + JavaScript)
- 인터넷 연결 시 Pretendard 폰트 자동 적용

---

## 데이터 저장 방식

- 브라우저 `localStorage` 사용 (키: `dalcom_v1`)
- 새로고침·재실행 후에도 입력 내용 유지
- 초기 실행 시 샘플 데이터 21건 자동 삽입

## 디자인 가이드

- **폰트**: Pretendard (CDN) → Apple SD Gothic Neo → 시스템 sans-serif 폴백
- **포인트 컬러**: #4f46e5 (인디고)
- **배경**: #f8fafc (연한 회백색)
- **카드**: 흰 배경, 1px 테두리, 12px 둥근 모서리
- **반응형**: 960px 이하에서 2열 → 1열 전환

---

## 사용 방법

1. `dashboard.html` 파일을 더블클릭해서 브라우저로 열기
2. 상단 진행률 카드 또는 탭을 클릭해 부서 이동
3. `[일일 / 주간 / 월간]` 버튼으로 기간 전환
4. 우측 `[+ 업무 추가]` 버튼으로 신규 업무 등록
5. 카드 위에 마우스를 올리면 ✏️ (수정) / 🗑️ (삭제) 버튼 표시
6. 카드 내 상태 드롭다운으로 진행 상황 즉시 변경
